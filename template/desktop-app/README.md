# {{projectName}} — Nexus Desktop App

Generated by the Nexus Framework Client from the **desktop-app** template.
The flagship sample is a Desmos-style single-page function plotter: Python
(numpy) samples the curves, C++ owns the data, ImGui + ImPlot draw the page,
and Lua/TS-XHTML layers show how the UI is scripted and authored.

## Stack

| Layer | Tech | Where |
|-------|------|-------|
| Window / input / GPU | SDL3 (OpenGL 3.3) | `src/main.cpp` |
| UI widgets & chart | Dear ImGui + ImPlot (+ imnodes for the blueprint editor) | `src/view/` |
| Runtime scripting | Lua 5.4 via sol2 | `src/view/LuaPanels.*`, `scripts/panels.lua` |
| Authoring DSL | TypeScript + XHTML (lowers into Lua) | `ui/ui.xhtml`, `ui/ui.ts` |
| Math backend | Embedded CPython + numpy via pybind11 | `src/controller/PythonEngine.*`, `python/functions.py` |
| App graph | Langflow-style JSON, editable in the imnodes GUI | `blueprint.json` |

## Layout (MVC)

```
src/
├── model/        FunctionRegistry — function catalog + sampled-data caches
├── controller/   PlotController + PythonEngine (pybind11 embed)
└── view/         PlotterView (ImGui/ImPlot page) + LuaPanels (sol2)
```

## Build

Prerequisites: CMake ≥ 3.24, Ninja, a C++20 compiler, Python 3.10+ with dev
headers, and `pip install -r requirements.txt` into that interpreter.
SDL3/ImGui/ImPlot/imnodes/Lua/sol2/pybind11 are fetched automatically
(FetchContent); pass `-DNXS_PREFER_SYSTEM_DEPS=ON` to prefer system packages.

```bash
cmake --preset debug          # configures into ../../builds/framework/{{projectName}}/debug
cmake --build --preset debug
./../../builds/framework/{{projectName}}/debug/{{projectName}}
```

Explicit out-of-source configure (no presets):

```bash
cmake -B ../../builds/framework/{{projectName}}/debug -G Ninja -DCMAKE_BUILD_TYPE=Debug
cmake --build ../../builds/framework/{{projectName}}/debug
```

## Try it

- Pick a function in the sidebar combo and press **Add** — plot several at
  once, each with its own color/visibility controls.
- Drag to pan, wheel to zoom; tweak x-range, sample count (64–4096), grid,
  and the linear/log Y toggle.
- **F1** loads a demo curve set, **L** toggles log scale — both are wired in
  `scripts/panels.lua`, which you can edit and hot-reload.
- `blueprint.json` is the imnodes flow graph for this app; re-open it in the
  wizard to rewire the page.
