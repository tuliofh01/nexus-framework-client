# {{projectName}} — Nexus Desktop App

Generated by [The Nexus Framework](https://github.com/tuliofh01/nexus-framework-client) — a native app scaffold that turns visual blueprints into compiled C++/Lua/Python binaries. SDL3 + Dear ImGui, no browser, no Electron, no cloud.

---

## System structure

```
{{projectName}}/
├── build.zig              # Zig build graph — compile C++, link SDL3, pack archives
├── build.zig.zon          # Zig dependency manifest — pinned tarballs
├── nxs_config.json        # Project config — name, version, template, dependencies (v2 schema)
├── blueprint.json         # App structure graph (Langflow-style nodes + edges)
│
├── src/                   # C++20 MVC — your domain logic
│   ├── main.cpp           # SDL3 + ImGui bootstrap loop
│   ├── model/             # Durable state (AppModel, FunctionRegistry)
│   ├── controller/        # Commands + PythonEngine
│   └── view/              # ImGui screens + Lua panels
│
├── shared/                # Cross-target runtime helpers (theme, fonts, script archive)
├── python/                # Python analytics (functions.py) — pybind11 in-process
├── scripts/               # Lua panels (panels.lua) — sol2 hot-reloadable
├── ui/                    # TypeScript + XHTML — declarative UI lowering to ImGui
├── flows/                 # Runtime automations (flows.json) — triggers + steps
├── zig-services/          # Zig build sidecar
└── assets/                # Themes, fonts, logo
```

### MVC layers

| Layer          | Directory         | Responsibility                          | Language |
|---------------|-------------------|-----------------------------------------|---------|
| **Model**      | `src/model/`      | Domain state — RAII objects, registries | C++20    |
| **Controller** | `src/controller/` | Commands, orchestration, Python engine  | C++20    |
| **View**       | `src/view/`       | ImGui draw, Lua panel host              | C++20    |

## Implementation guide

### Adding C++ code

1. Create or edit files in `src/model/`, `src/controller/`, or `src/view/`
2. Add new `.cppm`/`.cpp` paths to `zig-services/build.zig`
3. Rebuild with `zig build`

Conventions: RAII for all resources, `std::span`/`std::string_view`, `[[nodiscard]]`, `constexpr`. No C-style casts. Run `clang-format` before commits.

### Adding Lua panels

Edit `scripts/panels.lua`:

```lua
nxs.register_panel("My Panel", function()
    if ui.button("Click me") then
        nxs.log("Button clicked")
    end
end)

nxs.register_hotkey(keys.F1, function()
    nxs.log("F1 pressed")
end)
```

Lua reloads at runtime without recompiling — scripts are packed into `misc/lua.dat` on build, but plaintext `scripts/` fallback is checked first during development.

### Adding Python analytics

Edit `python/functions.py`:

```python
def evaluate(func: str, x_min: float, x_max: float, samples: int):
    import numpy as np
    xs = np.linspace(x_min, x_max, samples)
    ys = eval(func)(xs)
    return xs, ys
```

Python runs in-process via pybind11 — no serialization, no IPC. numpy arrays cross via buffer protocol. Install dependencies: `pip install -r requirements.txt`.

### Adding TypeScript/XHTML UI

Edit `ui/ui.xhtml` for structure, `ui/ui.ts` for bindings:

```xml
<panel id="sidebar" title="Functions" width="320">
    <combo id="function-picker" items-source="availableFunctions" bind="pendingFunction"/>
    <button label="Add" on-click="addPending"/>
</panel>
```

The toolchain lowers this to Lua/ImGui calls at build time.

### Editing the blueprint

`blueprint.json` defines your app's MVC wiring as a Langflow-style graph. Nodes: `python.module`, `cpp.model`, `cpp.controller`, `ui.page`, `lua.script`. Edges: data ports (`sampleCache`) and command ports (`commands`).

### Adding runtime flows

Enable `flows/flows.json` in `nxs_config.json`:

```json
{ "flows": { "enabled": true, "max_scheduled": 10, "max_concurrent": 4 } }
```

Flow modes: `background` (interval), `triggered` (event + condition), `startup`, `manual`.

## Cross-language bridge

C++, Python, Lua, and Zig can call each other in-process:

| From ↓     | To → C++              | To → Python              | To → Lua       | To → Zig              |
|-----------|-----------------------|--------------------------|----------------|----------------------|
| **C++**    | Direct                | `PythonEngine`           | `LuaPanels`    | C ABI (`nxs_*`)       |
| **Python** | `call_cpp("fn")`      | Direct                   | Via C++ bridge | Via C++ bridge        |
| **Lua**    | `nxs.bridge.call_cpp` | `nxs.bridge.call_python` | Direct         | `nxs.bridge.call_zig` |
| **Zig**    | C ABI `nxs_*`         | Via C++ bridge           | Via C++ bridge | Direct                |

## Debugging

```bash

# Run with address sanitizer
zig build -Doptimize=Debug

lldb ./zig-out/bin/{{projectName}}

# Check all languages are linked
./zig-out/bin/{{projectName}} --version
```

### Common issues

| Issue                         | Likely cause                         | Fix                                 |
|------------------------------|--------------------------------------|------------------------------------|
| `SDL_Init failed`             | Missing SDL3 runtime                 | `apt install libsdl3-dev`           |
| `PythonEngine: import failed` | Missing Python package               | `pip install -r requirements.txt`   |
| `Module not found`            | `.cppm` path missing in `build.zig`  | Add source to module sources list   |
| `Lua panel not showing`       | Script error in `panels.lua`         | Check console output for Lua errors |
| `Linker error: undefined`     | `.cpp` not registered in `build.zig` | Add to app/module sources           |

## Themes & icons

Set theme in `nxs_config.json` or at startup in `main.cpp`:

```cpp
nxs::runtime::NexusTheme::applyFromFile("assets/themes/nexus-dark.json");
```

Presets: `nexus-dark`, `nexus-light`, `nexus-field` (in `assets/themes/`).

Nerd Font icons: download JetBrainsMono Nerd Font to `assets/fonts/NexusNerdFont-Regular.ttf`.

## Adding dependencies

| Type   | How                                                           |
|-------|--------------------------------------------------------------|
| Python | `pip install <pkg>` then add to `requirements.txt`            |
| Lua    | sol2 bindings — add C++ wrapper in `src/`                     |
| C++    | Add system lib via `zig build` or zig package via `zig fetch` |
| Zig    | `zig fetch --save <url>` into `build.zig.zon`                 |
