## Translation guidelines

- **Full translations** should mirror all sections of the English README, including the table of contents, architecture overview, generation pipeline, build guide, comparisons, C++ deep-dive, and community sections.
- **Concise translations** (es, de, ru, zh-CN) intentionally cover only the essentials: a quick start command sequence, documentation links, the license, and the tagline. This keeps maintenance low while giving non-English readers a landing page and a path to the English docs.
- The `pt-BR` translation is the only full translation maintained in this repo. PRs to expand other languages to full coverage are welcome.
